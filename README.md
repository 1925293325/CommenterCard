# CommenterCard —— 评论者信息卡片插件

> 把鼠标移到评论区头像上，看看谁是「话痨」、谁是「萌新」~ 🎭

插件作者是 [Lin.](https://linyu.live)，用 Typecho 写博客的朋友可以试试这个小玩意。

## 它能干啥

一句话：鼠标悬停评论区头像，弹出一张信息小卡片。

卡片上会显示：

- 🏷 **等级称号** —— 根据评论次数自动给头衔（萌新 → 活跃 → 元老 → 话痨 → 传说）
- 📍 **IP 属地** —— "IP 广东" 这种格式
- 🙉 **最新文章** —— 显示对方最近的三篇文章
- 💬 **最近评论** —— 如果获取不到文章，显示对方本站评论
- 🟢 **在线状态** —— 对方的博客能不能打开
- 🔗 **一键访问** —— 有填网址的就多一个访问按钮
- ✨ **博主标签** —— 后台填上你的邮箱，你的评论会显示「博主」蓝标

## 装一下

1. 把整个 `CommenterCard` 文件夹丢进 `usr/plugins/`
2. 后台 → 控制台 → 插件 → 启用
3. 点「设置」按需改改就行

> 内置了一张淡蓝水彩背景图，装完直接用，不想折腾的话啥也不用改。

## 后台可以调啥

**等级称号规则**  
默认是：
```
0,访客
10,活跃
50,元老
100,达人
200,大神
```
想改成什么就改什么，一行一条，格式是 `评论数,称号`。比如你觉得「达人」不好听，改成「社交达人」也行 😂

**卡片背景图片 URL**  
内置了一张牛皮纸色背景图柔和暖色调，覆盖整张卡片。想用自己的图就填个地址替换掉。

**背景图透明度**  
默认 15，数字越大图越明显。建议 15~30 之间，看得见图又不挡字。

**博主邮箱**  
填上你的邮箱（要跟你在博客里用的那个一致），自己的评论就会显示蓝色「博主」标签~

## 注意事项

- 需要主题评论区头像有 `class="avatar"`，大部分主题都有
- 服务器要能访问 `ip-api.com` 才能查 IP 属地（国内服务器可能不行，到时候自己换个 IP API 地址就行）
- 只会在文章页生效，首页不弹
- 暗色模式会跟着你的主题一起切换
